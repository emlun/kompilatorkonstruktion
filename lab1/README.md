Running
---

To run the tweet aggregation/sorting program (part 4 of the lab), run

    $ ./gradlew run

To run the tests, run

    $ ./gradlew test

To build a zip file to submit to the TA, run

    $ ./gradlew assemble

The zip file is placed in `build/distributions`.
